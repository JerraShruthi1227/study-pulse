# 🚀 StudyPulse — Smart Study Streak Tracker

A modern productivity web app designed to help students build **consistent daily learning habits** through streak tracking and visual motivation.

🔴 **Live App:** https://study-pulse.vercel.app

---

## ⭐ Problem Statement

Many students start learning with motivation but fail to stay consistent.
There is a lack of **simple habit-building tools focused on daily study discipline**.

StudyPulse solves this by creating a **visual streak psychology loop** that encourages users to maintain learning momentum.

---

## ✨ Key Features

✅ Daily Study Streak Tracking
✅ Total Study Days Counter
✅ Linear Study History Timeline
✅ Smart Streak Reset Logic
✅ Minimal Modern SaaS UI
✅ Real-time State Feedback
✅ Backend API Based Streak System
